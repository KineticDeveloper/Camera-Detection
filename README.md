# Camera Detection

## About the app

An app developed using Apple's SqueezeNet CoreML with ability to detect different objects using camera and play back using siri

## Main Fnctionality

- CoreML integration
- Photo captureing, Analysing and presenting
- Speak out result
- Flash controll integration
